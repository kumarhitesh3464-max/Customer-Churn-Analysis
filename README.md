# 📊 Customer Churn Analysis

## 📌 Overview

This project predicts customer churn using machine learning. It analyzes customer behavior to identify customers who are likely to leave the service and provides business insights for improving customer retention.

## 🎯 Objectives

* Analyze customer churn patterns
* Perform data cleaning and preprocessing
* Build a Logistic Regression model
* Evaluate model performance
* Generate business insights

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## 📂 Dataset

Telco Customer Churn Dataset

## 📈 Results

* Built a Logistic Regression model for churn prediction.
* Identified key factors affecting customer churn.
* Generated insights to improve customer retention strategies.

## 🚀 Future Improvements

* Random Forest & XGBoost
